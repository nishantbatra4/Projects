------------------------------------------------------------------------


Drone Aided Agriculture helps in creating a medium of communication 
between the farm owner and the managers with the drone in order to carry
forward various drone functionalities and to get maximum yield.
------------------------------------------------------------------------




PROJECT TITLE: Drone Aided Agriculture
PURPOSE OF PROJECT: To achieve smart farming using a drone. 
VERSION OF DATE: Version 1.0 10/21/2018
HOW TO START THIS PROJECT: Run the Main class to start the application. See instructions below for system requirements. 


AUTHORS: Caren Alexander, Nishant Batra, Jiajun Lin, J Wang
USER INSTRUCTIONS:
Requires Java JDK 8 as well as JavaFX SDK Bundle to be run successfully. JavaFX needs to be enabled in the IDE Plugins. The application will be run from the Main class.
